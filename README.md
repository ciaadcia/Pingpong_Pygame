# Pingpong Game
Permainan Pong sederhana yang dibuat menggunakan Pygame, di mana dua pemain mengendalikan paddle (raket) untuk memantulkan bola agar tidak keluar dari area permainan.

# Cara bermain:

Player 1 (kiri) menggunakan tombol:
- W untuk bergerak ke atas
- S untuk bergerak ke bawah

Player 2 (kanan) menggunakan tombol:
- ↑ (panah atas) untuk bergerak ke atas
- ↓ (panah bawah) untuk bergerak ke bawah

Bola akan bergerak otomatis dan memantul saat mengenai paddle atau dinding atas/bawah.
Jika bola melewati paddle:
- Player 1 kalah jika bola keluar dari sisi kiri
- Player 2 kalah jika bola keluar dari sisi kanan

Game akan menampilkan pesan kekalahan saat salah satu pemain gagal menahan bola.
